Getting Started

A starting point to build on top of Cardano is having a good understanding of configuring, running and interacting with the Cardano node. Having this base knowledge is essential whether you want to build Dapps, chain analysis tools, wallets, mint custom tokens or operate your own stake pool.

Set up a Linux Environment

At the time of writing, running the Cardano node in Linux is the easiest and most well-documented approach. There are three ways to do so.

  • Dual boot a Linux distribution on top of your default OS
  • Run a Linux distribution as using WSL2 (Windows)
  • Run a Linux distribution as a local VM
  • Run a Linux distribution as a cloud VM via SSH

The most important factor is to have a repeatable and reliable way of setting up the environment.
